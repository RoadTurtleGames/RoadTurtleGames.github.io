Detailed Description

Base class that provides a common implementation for simple effects that do not require object pooling.

Template Parameters
TEffectAuthoringThe type for the script used for effect authoring.
TEffectThe type for the script used for the effect data suitable for jobs.
TEffectResultThe type for the script used for the effect result suitable for jobs.
