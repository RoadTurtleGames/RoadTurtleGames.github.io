Installation

Impact CFX requires the Burst, Collections, and Mathematics packages. These are included by default in most new projects, but if they are not in your project you can manually install them using the Package Manager.

Other than this, Impact CFX does not require any special installation steps to get you up and running. However, there is a Gizmos folder included in the package that contains icons that you can install into your project if you want to have special icons for the Impact CFX components and assets.

To install the icons, simply move the Gizmos folder to the root of your project. If there is already a Gizmos folder in your project's root, move the Impact CFX folder within the Gizmos folder into the existing folder.
