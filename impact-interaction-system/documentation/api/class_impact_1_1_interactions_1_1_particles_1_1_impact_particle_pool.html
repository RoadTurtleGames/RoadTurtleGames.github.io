Parameters
interactionResultThe interaction result to use data from.
pointThe world position to emit the particles at.
normalThe surface normal used to set the particle's rotation.
priorityThe priority of the particles. Particles with a priority less than this will be "stolen" and used for this result.
Returns
The ImpactParticles that will be used to emit the particles. Will be null if no particles are available.
