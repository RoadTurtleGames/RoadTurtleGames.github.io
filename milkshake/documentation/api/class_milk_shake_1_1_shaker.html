static

Starts a shake for all Global Shakers, using a position and distance to scale the strength of the shake. If the distance between the shaker and the point is greater than the max distance, no shake will be applied.

Parameters
pointThe poisition that the shake originated from.
maxDistanceThe maximum distance the shake should have an affect.
shakeDataThe shake parameters such as strength and roughness and fade in/out times.
shakeInstancesAn optional list that will be populated with all of the created ShakeInstances.
seedA optional seed to use for the shake. Shakes that have the same seed will have the same movement.

If the distance between this Shaker and the point is greater than the max distance, null will be returned.
